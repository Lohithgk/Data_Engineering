# Data_Engineering
A project demonstrating extract, transform, and load (ETL) processes using a star schema design, PySpark and SparkSQL.

# Data Model
Fact Table:
- Sales.CSV

Dim Tables:
- Products.CSV
- Product_Subcategories.CSV
- Product_Categories.CSV
- Territories.CSV
- Customers.CSV
